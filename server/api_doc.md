# Todo App Server
Website for manage your todo. This app has : 
* CRUD Todo
* JSON formatted response

&nbsp;

## Endpoints
```
- GET /todos
- POST /todos
- GET /todos/:id
- PUT /todos/:id
- DELETE /todos/:id
- POST /register
- POST /login
```

## RESTful endpoints
### GET /todos

> Get all todos

_Request Header_
```json
{
  "access_token": "<your access token>"
}
```

_Request Body_

### POST /todos

> Create new todo

_Request Header_
```json
{

}
```

_Request Body_
```json
{
    "title": "<todo title>",
    "description": "<todo description>",
    "status": "<todo status>",
    "due_date": "<todo due_date>"

```
