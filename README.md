# fancy-todo
A Todo-List App, Made With Express

# Endpoints
```
- GET /todos
- GET /todos/:id
- POST /todos
- PUT /todos/:id
- DELETE /todos/:id
- POST /register
- POST /login
- GET /movies
```

# POST /todos
> Create A New Todo
Request Header
```
{
  "access_token": "<your access token>"
}
```

Request Body
```
{
    "title": "<todo's name>",
    "description": "<todo's description>",
    "status": "<todo's status>",
    "due_date": "<todo's due_date ( Must Not )>"

}
```
