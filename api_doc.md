# My Fancy Todo App Server
My Fancy Todo App is an application to manage your activities. This app has : 
* RESTful endpoints for todo's CRUD operations
* JSON formatted response

&nbsp;

## Endpoints
```
 - POST /register
 - POST /login
 - POST /googleLogin
 - POST /todos
 - GET /todos
 - GET /todos/:id
 - PUT /todos/:id
 - DELETE /todos/:id
```

## RESTful endpoints
### POST /register

> Register user

_Request Header_
```
not needed
```

_Request Body_
```
{
  "username": "johndoe",
  "email": "johndoe@mail.com",
  "password": "johndoe123"
}
```

_Response (201 - Created)_
```
{
  "id": "1",
  "email": "johndoe@mail.com"
}
```

_Response (400 - Bad Request)_
```
[
  "<error message 1>",
  "<error message 2>",
  ...,
  "<error message n>"
]
```

_Response (500 - Internal Server Error)_
```
[
  "<error message>"
]
```
